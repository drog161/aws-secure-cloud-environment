# AWS Secure Cloud Environment
## Project Overview

This project demonstrates the design and deployment of a secure AWS cloud environment using AWS best practices.

## Technologies Used

- AWS VPC
- EC2
- CloudTrail
- CloudWatch
- Security Group

## Architecture

- Public Subnet
- Private Subnet
- Internet Gateway
- Route Tables
- Security Groups
- IAM Roles

## Security Features

- Least privilege IAM permissions
- CloudTrail auditing
- CloudWatch monitoring
- Network segmentation
- Controlled inbound and outbound traffic

## Skills Demonstrated

- AWS Cloud Architecture
- Cloud Security
- Identity and Access Management
- Network Design
- Troubleshooting

## Lessons Learned

This project improved my understanding of AWS networking, IAM permissions, logging, monitoring, and cloud security best practices while preparing for the AWS Certified Solutions Architect - Associate certification.
